
# Arch Angels

A advanced utilities bot.


## Features

- Rich Presence
- Online Status
- Activity
- Send Messages
- Latency (ms)


## Installation

Install bot with python

```bash
  cd ArchAngelsBOT
  pip install discord
```
    
## Change your bot token

```
bot.run('[TOKEN]')
```
## Deployment

To deploy this project run

```bash
  python bot.py
```
